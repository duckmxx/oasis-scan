⚡ Quick Start — Desktop GUI
1
Install dependencies
# Arch Linux
sudo pacman -S python tk

# Debian / Ubuntu
sudo apt install python3 python3-tk
2
Run the GUI
python3 gui.py
3
Sign in with your Oasis account
Use the same email and password as this dashboard. The scan results will appear in Devices automatically.
⌨ Quick Start — CLI (headless / SSH)
1
Download both files to the same directory
curl -O https://scanoasis.tech/download/scanner.py
curl -O https://scanoasis.tech/download/gui.py
2
Run the CLI scanner directly
python3 scanner.py
3
Optional: schedule with cron
# Run every day at 3 AM
0 3 * * * python3 /opt/oasis/scanner.py
